NoSQL alternatives!!!

In theory The original intention has been modern web-scale database. The movement Next Generation Databases mostly addressing some of points: being non-relational, distributed, open-souce and horizontally scalable. by nosql-database.org

There are Mainly four type of database
  • Key Value Pair
  • Document Store
  • Wide Column Store
  • Graph Store

Some of the popular key value store providers are


  • DynamoDB
  • Azure Table Storage
  • Riak
  • Redis

Some of the popular document store providers are


  • MongoDB
  • CouchDB
  • RavenDB

Some of the popular wide column store providers are


  • hadoop
  • cassandra
  • hypertable
  • amazon simpledb

Some of the popular graph database store providers are


  • Neo4j
  • Infinite Graph
  • Sones
